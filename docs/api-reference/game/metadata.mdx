## Getting Metadata

### Endpoint

```http
GET /api/game/metadata/get
```

<ParamField header="authorization header" type="string" required>
    Bearer [token]
</ParamField>

### Example Response

```json
{
    "data": "metadata"
}
```

## Adding & Updating Metadata

### Endpoint

```http
PUT /api/game/metadata/add
```

<ParamField header="authorization header" type="string" required>
    Bearer [token]
</ParamField>

<ParamField body="data" type="string" required>
    The metadata to store or update. must be 1000 characters or less.
</ParamField>

### Example Response

```json
{
    "data": "metadata"
}
```

## Deleting Metadata

### Endpoint

```http
DELETE /api/game/metadata/delete
```

<ParamField header="authorization header" type="string" required>
    Bearer [token]
</ParamField>
