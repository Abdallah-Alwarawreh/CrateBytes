## Submit Score

The `Submit Score` endpoint allows you to submit a player's score to a specific leaderboard.

### Endpoint

```http
POST /api/game/leaderboards/[leaderboard id]/submit
```

<ParamField body="score (body)" type="string" required>
    The score to submit to the leaderboard. must be passed in the request body.
</ParamField>

<ParamField header="authorization (header)" type="string" required>
    Bearer [token]
</ParamField>

### Example Response

```json
Score submitted
```

## Get leaderboard data

The `Get Leaderboard Data` endpoint allows you to retrieve the data for a specific leaderboard, including player scores and pagination information.

### Endpoint

```http
GET /api/leaderboards/[leaderboard id]?page=[page number]
```
